A voice cloning tool built on ResembleAI's Chatterbox TTS model. Requires only sample audio files for zero-shot cloning — no fine-tuning needed. Supports emotional control via exaggeration and pacing parameters, preset modes (dramatic, calm), and real-time interactive synthesis. Optimized for NVIDIA Blackwell (RTX 50xx) with PyTorch 2.8 + CUDA 12.8.
