Koji is an emotionally-aware robot combining a Go backend brain server with a C++ ESP32 face display module. The system features a full emotional state machine with decay cycles — moods shift dynamically based on interactions and idle time. The face display renders emotion-mapped expressions in real-time via an event-driven API. Deployed via Docker with CI/CD through Gitea Actions.
